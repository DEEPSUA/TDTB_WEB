Template Degree Telegram Bot¶

Documentación del bot de Telegram a modo de plantilla para la gestión de grados.

Para poder usar la plantilla se tendrá que descargar la última versión del proyecto, rellenar el archivo de las variables de entorno (.env) y revisar las funcionalidades para habilitar o deshabilitar las que sean necesarias. O crear nuevas. Además de esto, será crucial instalar las dependencias del bot de Telegram en nuestro sistema para poder lanzarlo y testear su funcionamiento.

Variables de entorno¶

Las variables de entorno del bot de Telegram se escriben en el archivo .env, que no está incluído en el repositorio porque es un archivo protegido. Aun así, tenemos el archivo .env.example con algunas variables ya rellenadas que se recomienda no cambiarlas, puesto que implicaría un cambio en gran parte del proyecto.

Funcionalidades¶

Toda la parte técnica está muy bien, pero yo quiero saber qué puede hacer el bot.

Una vez configurado el archivo .env y lanzado el bot, podremos hacer uso de los siguientes comandos:

Comando Estado Descripción
/admins OK Acceso a las funcionalidades de administradores
/calendarios OK Te envía los enlaces a los calendarios de entregas y exámenes
/coordi OK Te dice quién es el coordinador o coordinadora del grado
/delegadxs OK Te dice quienes son los representantes de grupo del grado
/grupos OK Te devuelve el listado de grupos y canales del grado
/materias OK Te devuelve el listado de los grupos de las materias del grado
/rrss OK Redes sociales de la DEEPSUA
/addadmin OK Añade un administrador al bot.
/savegroup OK Guarda el grupo actual a la base de datos del bot

Funcionalidades para administradores¶

El comando /admin nos abre un menú de botones en el que podemos seleccionar entre las distintas funcionalidades de administradores. Estas funcionalidades son las siguientes:

Funcionalidad Estado Descripción
Admins - Eliminar OK Abre un menú de botones con los administradores del bot. Seleccionas al que quieres eliminar y lo elimina.
Admins - Ver OK Permite ver los administradores del bot
Difundir OK Difunde un mensaje por los canales
Grupos - Salir OK Abre un listado de botones con los grupos a los que ha entrado. Seleccionas un grupo y el bot sale de él.
Grupos - Ver OK Permite ver los grupos a los que el bot ha accedido

Dependencias¶

Las principales dependencias del bot serán algunas de las herramientas más útiles y básicas de Python, además de la librería Python-Telegram-Bot. Podemos encontrar las dependencias en el archivo requirements.txt:

pytz
python-dotenv
python-telegram-bot

Para poder instalar las dependecias y no hacerlo una por una, usaremos el siguiente comando:

pip install -r requirements.txt
